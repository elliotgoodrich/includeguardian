IncludeGuardian

IncludeGuardian is a free tool to help improve C and C++ build times. The most time consuming part of C++ build times is usually not compiling or linking, but in preprocessing and syntax checking files that have been #includeed. A simple "Hello World" application in C++ will be preprocessed to almost a 1MB file from only the #include <iostream> line. This is all run through the C++ compiler and will take an additional 150ms to check.

IncludeGuardian makes recommendations for your project on how to reduce work the compiler needs to do. These suggestions include:

  • What files to add to a precompiled header
  • Which #include directives are the most expensive
  • Which header files should be split up
  • Which header+source pairs are unused
  • Which header+source files should be combined to be header-only
  • Which files have an incorrect include guard

With each suggestion IncludeGuardian will give a percentage saving for the number of preprocessing tokens that need to be compiled. Although this is not a perfect substitute for the actual time saved, it is a reasonable alternative that will give a deterministic result in a fraction of the time.
